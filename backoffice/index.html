운영 메모

이 목업 백오피스의 저장 방식과 폴더 구조를 빠르게 확인할 수 있습니다.

영역 설명
backoffice/app.js 모든 편집기 공통 저장, 불러오기, JSON 내보내기를 담당합니다.
backoffice/style.css 백오피스 전용 공통 레이아웃과 폼 스타일을 관리합니다.
content-data/*.json 각 편집기의 기본 시드와 공용 반영용 site-content.json 구조를 보관합니다.
localStorage 편집 중 임시 저장은 브라우저별 로컬 저장소에 보관되고, 배포용 공용 반영은 site-content.json으로 내보냅니다.
